CSIR Intelligent Manipulator
----------------------------

Robotic manipulation system being developed at MIAS (Mobile Intelligent Autonomous
Systems) in the CSIR.

Organisation
------------
csir_wam_arm_navigation
  Automatically generated by ROS arm navigation wizard
csir_wam_common
  Robot description
csir_wam_controllers
  Packages for interfacing between planners and robot drivers
csir_wam_object_manipulation
  Packages for object manipulation (e.g. pick & place)
